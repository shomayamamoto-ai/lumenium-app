# /rehearse — 面談ロールプレイ生成

面談前 10 分の **壁打ち練習** 用に、ブリーフから想定される対話の流れを台本化します。

## 引数
- `$1` : 候補者 slug（`brief.md` が必須）

## 手順

### Step 1: ブリーフ読み込み
- `candidates/$1/brief.md` を Read
- 無ければ「先に /brief $1 を」と案内

### Step 2: 面談 30 分の流れを 5 フェーズに分けて台本化

#### フェーズ 1（0–3 分）: アイスブレイク
- 候補者の SNS 投稿を褒める実在のネタで切り出す
- 候補者の想定反応を 2 パターン（「明るく返す」「照れて最小限」）

#### フェーズ 2（3–10 分）: 動機ヒアリング
- オープンクエスチョン 3 つ（「なぜモデルを?」「いつから?」「他に何を検討してる?」）
- 各問いに対する **候補者の想定回答 A/B** と、**スカウト側の深掘り次手**

#### フェーズ 3（10–20 分）: 現在地と目標ギャップ
- 候補者の自己評価を引き出す問い
- ブリーフで検出した「動機の仮説」を検証する問い
- 想定される自己認識の歪み（過大評価 / 過少評価）と、それへの返し

#### フェーズ 4（20–27 分）: 訴求＆断り想定
- ブリーフで予測した **3 つの不安・反論** それぞれについて、候補者が口にする想定台詞と、こちらのベスト返し
- 「すぐに決めてくれない」ケースの対処

#### フェーズ 5（27–30 分）: クロージング
- 決裁者確認（「ご家族への相談は?」等）
- 次アクションの握り（「体験レッスン / 資料送付 / 後日決定」のどれに落とし込むか）
- 必ず聞くべき質問（予算、希望曜日、連絡手段）のチェックリスト

### Step 3: 「絶対に言ってはいけないこと」リスト
候補者のプロファイルに応じた NG ワード 3–5 個を明示:
- 例（未成年疑いあり）: 「親には内緒で」の類は絶対 NG
- 例（業界不信が強い）: 過度な成功事例の羅列（逆効果）
- 例（迷い層）: 「今決めないと席が…」の煽り（後で揉める）

### Step 4: セルフ採点シート
面談後に自分を振り返るチェックリスト 10 項目:
- [ ] 相手の発言を逐語でメモした（具体発言が 3 つ以上ある）
- [ ] 予算感をヒアリングできた
- [ ] 決裁者を確認した
- [ ] 次アクションを握った
- [ ] 「断り」が出た時に一度受け止めてから返した
- [ ] 相手の Yes/No を決めつけず、選択肢を提示した
- [ ] 未成年の場合、保護者同意の話をした
- [ ] こちらの話が 50% を超えていない
- [ ] 沈黙を 3 秒以上許容した
- [ ] 面談後の期日を相手と合意した

### Step 5: 保存と返却
- `candidates/$1/rehearsal.md` に Write
- ユーザーには「5 フェーズの見出しだけ」を表示し、「詳細は rehearsal.md を読み、5 分黙読してから面談に臨むのがおすすめ」と添える

## 制約
- 想定台詞はすべて「仮」であることを明示（「以下は架空の想定」）
- 未成年 or 保護者同席必要なケースでは、クロージングに必ずその旨の台本を含める
- 相手を操作する手法（焦らせ、虚偽の希少性 等）は書かない
