# /followup — 面談後フォローメッセージ生成

`/classify` の結果と議事録を読み、**分類と障壁に応じた** メッセージ文面を生成します。TODO に「家族向け資料を送る」と書かれるだけでなく、**実際に送れる文面まで** 作ります。

## 引数
- `$1` : 候補者 slug

## 手順

### Step 1: 情報収集
- `candidates/$1/minutes.md` を Read（必須）
- `candidates/$1/classification.md` を Read（必須、なければ「先に /classify を」と案内）
- `candidates/$1/brief.md` を Read（トーン・関心を参照）

### Step 2: 分類ごとに文面を生成

#### 🔴 本気層
以下の **3 メッセージ** を生成:

1. **即送信メッセージ（24h 以内）**
   - 面談の御礼 + 具体契約案内 + 決済リンク案内
   - 候補者が面談で刺さった訴求を最低 1 つ引用
   - 長さ 300–450 字

2. **契約書/料金案内の本文**
   - プラン、開始日、支払い方法を明文化
   - 質問受付の窓口を明示

3. **3 日後リマインダー**
   - 返信なかった場合に送るソフトな push
   - 「何か不安な点あれば気軽に」型

#### 🟠 迷い層
以下の **4 メッセージ** を生成:

1. **24h 以内の障壁打ち消しメッセージ**
   - 議事録から **最大の障壁** を抽出し、それ専用の文面
   - 障壁別テンプレ:
     - 家族未相談 → 家族向け説明資料の案内 + 同席面談オファー
     - 価格感度 → 分割プラン提案 + 実質月額換算
     - 時間の不安 → 週 1 プラン / オンライン併用の案内
     - 業界不信 → 合格者事例・契約先実名（公開許諾済みのみ）

2. **7 日後のナーチャリング**
   - 合格者インタビュー / SNS 成長事例を軽く共有

3. **14 日後の体験レッスン招待**

4. **30 日後の最終フォロー**（失注判定タイミング）

#### 🟢 冷やかし層
- 単発メッセージ 1 通のみ（体験レッスンいつでも型）
- 以後の自動フォローはしない旨、内部メモに残す

### Step 3: 各メッセージに必須要素を含める
- [ ] 候補者の **固有情報**（議事録の逐語引用を 1 つ使う）
- [ ] **次アクションが明確**（返信、日程提示、資料確認 等）
- [ ] **期限感**（「今週中に」「○月○日まで」等、焦らせない範囲で）
- [ ] **オプトアウト導線**（無理強い感を削ぐ「難しければ気兼ねなく」等）

### Step 4: 送信チャネル別バージョン
主要メッセージは以下 2 版を用意:
- **LINE/DM 版**: 150–250 字、絵文字最大 2 個、砕けた口調
- **メール版**: 件名付き、350–500 字、丁寧

### Step 5: 保存と返却
- `candidates/$1/followup-messages.md` に全パターンを Write
- ユーザーには **即送信メッセージの LINE 版だけ** を表示 + 「他のパターンは `followup-messages.md` 参照」

### Step 6: Gmail 下書き作成の提案（MCP 接続時のみ、任意）
Gmail MCP が接続されている場合、ユーザーに以下を問う:
```
即送信メッセージを Gmail 下書きとして作成しますか? (Y/n)
※ 自動送信はしません、下書き保存のみ
```
Y なら `/email-draft $1 immediate` を内部的に実行。
N またはスキップ時は「必要なら `/email-draft <slug>` でいつでも下書き作成できます」と案内のみ。

## 出力例
```
📬 フォローメッセージ生成: candidates/2026-05-01-tanaka-yuki/followup-messages.md

分類: 迷い層（本気寄り）
最大障壁: 家族未相談

━━━ 即送信（LINE 版）━━━
田中さん、本日は面談の時間をありがとうございました🙌
「3 年後に CM 1 本」という具体的な目標、本気で叶えにいける目標だと感じています。
ご家族への説明についてお話があったので、保護者の方向けの資料を PDF でお送りしますね。
内容は 5 分で読める形にしてあるので、週末にでも見てもらえたらと思います。
もし家族と一緒にもう一度面談することも可能ですので、気兼ねなく教えてください！

その他パターン（メール版・7 日後ナーチャリング・14 日後招待・30 日最終）は
followup-messages.md をご覧ください。
```

## 制約
- 候補者の発言を引用する際は **議事録の発言ログから実在のものだけ** 使う（捏造禁止）
- メッセージは **ユーザー（スカウト担当）の名前で送る想定**で一人称は書きすぎない
- 法的拘束や契約確定を示唆する表現（「契約成立しました」等）は禁止
- 未成年 or 保護者同意必要と判定されている場合、全メッセージに保護者同席の選択肢を明示
