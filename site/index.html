About Us

私たちについて

ゼミの目的

慶友会 社会学ゼミは、慶應義塾大学 通信教育課程で学ぶ学生による、自発的な学習サークルです。個々に学ぶ通信生が集い、講師を囲むゼミ形式で、本質的な学びを深めます。

ゼミ形式の学び

慶應義塾の「自我作古」の精神のもと、文・法・経済の領域を横断し、講師派遣による対話型のゼミを展開しています。

全国どこからでも

時間の制約や遠方にお住まいの方にも学びが届くよう、Zoomによる中継とスライド共有を活用しています。（教員・大学の許可のもと運営）

03

Philosophy

社会学という視座

⁂
秩序と進歩
社会学は、フランス革命後にオーギュスト・コントが
「社会物理学」として構想した学問である。
歴史・心理・経済を統合し、実証主義の科学的アプローチで
社会の「秩序と進歩」に貢献する——。
I
秩序 / Order

社会を成り立たせる構造と規範を、歴史的・実証的に読み解きます。

II
進歩 / Progress

AI・自動化時代に、人はどう生き、どう学ぶべきかを問い直します。

III
自我作古

前例なき時代を、自ら切り拓く人になる——慶應義塾の精神を体現します。

05

Current Themes

いま探究しているテーマ

毎期、参加者の関心と社会的問題意識を起点に、社会学の核となる主題を選定して掘り下げます。ここでは、現在進行中の探究テーマをご紹介します。

No.01

ハビトゥスと社会階層の再生産

ピエール・ブルデューの理論を出発点に、教育・文化・職業選択の中で 「階層」がいかに静かに継承されるかを読み解きます。

社会階層 文化資本
No.02

監視社会と「自由」の再定義

フーコーのパノプティコンから現代のプラットフォーム監視まで。 可視化される個人の中で、私たちが守るべき「自由」とは何かを問います。

権力 プライバシー
No.03

AI時代の労働と人間の尊厳

自動化が進む社会で、仕事を通じた自己実現はどう変わるのか。 マルクスの疎外論と現代の知識労働を接続して考えます。

労働社会学 技術
No.04

共同体と「弱い紐帯」の力

グラノヴェッターの理論を手がかりに、強い絆だけでなく 「弱い結びつき」が人生・キャリア・社会に果たす役割を再評価します。

ネットワーク 共同体

— Our Manifesto

学びは、 急がず。 本質に、近づく。

わかった気になる速さよりも、
本当に腑に落ちる深さを。
一人ひとりが、自分の問いを持って帰れる時間を、
大切にしています。

How We Learn

学びの流れ

  1. 1
    テーマ設定

    社会学の古典・現代理論から、参加者の問題意識に響くテーマを選定します。

  2. 2
    図解資料の作成

    難解な概念もかみくだいた図解資料で、誰もが本質にアクセスできるよう準備します。

  3. 3
    ゼミ形式での対話

    講師を囲み、Zoomで全国の仲間と対話。問いを重ねることで理解が深まります。

  4. 4
    省察と応用

    自分自身の生活・仕事に引き寄せて省察し、卒論や日々の実践へとつなげます。

FAQ

よくあるご質問

Q 社会学の予備知識がなくても参加できますか？ +

はい、まったく問題ありません。指定書籍『大学４年間の社会学が10時間でざっと学べる』から丁寧に土台を築きます。「本質をかみくだいた図解資料」で、初学者でも迷わず学べる設計です。

Q 会費はかかりますか？ +

詳細は入会申込時にご案内いたします。まずはお気軽にお問い合わせください。

Q 通信教育課程の入学前でも参加できますか？ +

ご出願予定の方も歓迎します。慶應通信での学び方や、卒論までの道のりなども、先輩メンバーから直接お話を聞いていただけます。

Q 遠方に住んでいますが参加可能ですか？ +

ゼミはZoomでのオンライン開催が中心です。全国どこからでも、また海外にお住まいの方でも、時差が合えばご参加いただけます。録画共有もあるため、後からの視聴も可能です。

Q 多忙でも続けられますか？ +

お仕事や家庭との両立を大切にしています。参加頻度は柔軟に調整可能、配信録画・図解資料で非同期での学習もサポートしています。寄り添う姿勢を何より大切に運営しています。

Q 卒業論文の相談はできますか？ +

もちろん可能です。社会学領域で卒論を執筆される方はもちろん、他分野の方でも、社会学的な視座から助言を得られます。職業背景を共有する仲間がいることが、テーマ選定でも大きな力になります。
